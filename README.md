# Valentine Roland - Professional Violin Tuition Website

A beautiful, responsive website for Valentine Roland's face-to-face violin tuition service in Oxfordshire.

## About

Valentine Roland offers professional violin lessons with home visits throughout Oxfordshire and surrounding areas. This website showcases her services, teaching philosophy, and provides easy booking options for potential students.

## Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Modern Styling** - Warm, earthy color scheme with professional typography
- **Two Main Pages**:
  - Home page with music showcase and introduction
  - Dedicated tuition page with pricing, testimonials, and booking
- **Contact Integration** - Email and WhatsApp booking options
- **SEO Optimized** - Proper meta tags and descriptions

## Color Scheme

- **Primary Brown**: `#654321` (headings, buttons)
- **Golden Yellow**: `#F9C349` (accents, hover states)
- **Warm Cream**: `#F3F0E6` (backgrounds)
- **White**: `#FFFFFF` (cards, contrast sections)

## Pages

1. **index.html** - Main landing page with music showcase
2. **tuition.html** - Detailed tuition information and booking

## Tech Stack

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript (for interactions and animations)
- Font Awesome icons
- Google Fonts (DM Serif Display, Open Sans, Great Vibes)

## Deployment

This site is designed to work with GitHub Pages. Simply:
1. Push to your repository
2. Go to Settings → Pages
3. Select "Deploy from a branch" → "main"
4. Your site will be live at: `https://ad-francis.github.io/valentine-roland-website/`
